SaaS Core Template Changelog
============================

Purpose
-------
This file tracks the template evolution phase by phase.

Rules:
- Update this file at the end of every implementation phase.
- Keep entries factual: what changed, what was verified, and what remains external/not configured.
- Do not record secrets, API keys, credentials, real customer data, or private provider details.
- Keep product-specific work out of this changelog unless it affects the reusable template.
